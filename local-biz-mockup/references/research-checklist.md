# Business Research Checklist

Systematic checklist for gathering comprehensive intelligence about any local business.

## 1. Website Content

- [ ] Home page — tagline, value proposition, hero copy
- [ ] Services/offerings — full list with descriptions
- [ ] About page — history, founding year, mission, values
- [ ] Team/staff page — names, titles, photos
- [ ] Contact page — all contact methods, hours, address
- [ ] Portfolio/projects/gallery — work samples, case studies
- [ ] Testimonials/reviews page — client quotes
- [ ] Blog/news — recent activity, topics covered
- [ ] Footer — license numbers, certifications, social links

## 2. Contact Information

- [ ] Primary phone number
- [ ] Secondary/estimates phone
- [ ] Email address(es)
- [ ] Physical address vs mailing address
- [ ] Business hours (all 7 days)
- [ ] WhatsApp / messaging links
- [ ] Social media profile URLs

## 3. Google Business Profile

- [ ] Business name as listed
- [ ] Category
- [ ] Star rating and review count
- [ ] Recent reviews (note themes — quality, reliability, communication)
- [ ] Photos uploaded by owner
- [ ] Hours of operation
- [ ] Website URL listed
- [ ] Q&A section

## 4. Local Directories (check 10+)

- [ ] Google Business / Google Maps
- [ ] Yelp
- [ ] BBB (Better Business Bureau)
- [ ] Yahoo Local
- [ ] State contractor registrar (e.g., AZ ROC, CA CSLB)
- [ ] State corporation commission / Secretary of State
- [ ] City/county business license database
- [ ] Local chamber of commerce
- [ ] Nextdoor (business listing)
- [ ] MapQuest

## 5. National / Industry Directories (check 10+)

- [ ] Manta
- [ ] Buzzfile
- [ ] The Blue Book (construction)
- [ ] Procore Network (construction)
- [ ] BuildZoom (contractors)
- [ ] Levelset (construction)
- [ ] Angi (Angie's List)
- [ ] HomeAdvisor
- [ ] Houzz (home services)
- [ ] Thumbtack
- [ ] OpenCorporates
- [ ] LinkedIn (company page)
- [ ] Yellow Pages / YP.com

## 6. Licensing & Credentials

- [ ] State contractor license number and type
- [ ] License status (active/inactive)
- [ ] License issue date and expiration
- [ ] Qualifying party / individuals on license
- [ ] Bond information (amount, carrier, status)
- [ ] Complaint history (open cases, disciplined cases)
- [ ] Insurance status
- [ ] Professional certifications
- [ ] Industry association memberships

## 7. Reviews & Ratings

For each platform where reviews exist, record:
- [ ] Platform name
- [ ] Star rating
- [ ] Number of reviews
- [ ] Common praise themes
- [ ] Common complaint themes
- [ ] Response pattern (does owner respond?)

## 8. Business Details

- [ ] Legal entity name
- [ ] Entity type (LLC, Corp, Sole Prop)
- [ ] Year established / incorporated
- [ ] State of incorporation
- [ ] Estimated annual revenue
- [ ] Estimated employee count
- [ ] Number of locations
- [ ] Ownership details (woman-owned, minority-owned, veteran-owned)
- [ ] SIC / NAICS codes
- [ ] Service area / geography served

## 9. Competitive Context

- [ ] Similar businesses in same area (from Google Maps "People also search for")
- [ ] How competitors present themselves online
- [ ] What differentiates this business from competitors

## Output Format

Save findings to `{project}/business-research.md` with:
- Table of Contents with anchor links
- Tables for structured data (contact info, directory listings, reviews)
- Direct quotes only when factual (license numbers, addresses)
- Sources section with URLs at the end
